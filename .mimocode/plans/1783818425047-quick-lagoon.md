# Plan: Deploy SIMBA v2 (UI/UX Improvements) ke Vercel

## Situasi
- SIMBA (Next.js 16) sudah deploy di Vercel via GitHub
- Branch `main` ada 1 commit, tapi working directory sudah punya 15 file modifikasi + 7 file baru (UI/UX improvements)
- Tidak ada perubahan env variables
- **Tujuan**: Upload perubahan yang sudah ada tanpa mengubah apapun

## Approach
Karena project sudah terkoneksi **GitHub → Vercel**, cukup:
1. `git add .` — stage semua perubahan yang sudah ada
2. `git commit` — commit dengan message yang jelas
3. `git push origin main` — push ke GitHub
4. Vercel otomatis build & deploy versi baru

**Tidak perlu**: vercel deploy manual, re-upload, atau setup ulang apapun.

## Yang Akan Dilakukan
1. Stage semua file yang berubah (modified + untracked)
2. Commit message:
   ```
   feat(ui-ux): improve dashboard UI/UX with responsive design and new components
   ```
3. Push ke `origin/main`
4. Vercel auto-deploys dalam 1-3 menit

## Verification
- Cek Vercel dashboard untuk deployment status
- Buka production URL untuk memastikan UI/UX baru berjalan
